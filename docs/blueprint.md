# **App Name**: Pixel Archive

## Core Features:

- Project Gallery Display: Showcase projects in a grid layout, featuring thumbnails, titles, and categories.
- Detailed Project Pages: Dedicated pages for each project with extended descriptions, multiple images/videos, and technologies used, similar to https://althruist.fyi/archive.
- Project Categorization & Filtering: Allow users to browse and filter projects by defined categories (e.g., 'Web Dev', 'Design', 'AI').
- About & Contact Section: A straightforward section for an 'About Me' description and essential contact information like email and social media links.
- Responsive Layout Adaptation: Ensure the portfolio dynamically adjusts and renders optimally across various devices (desktop, tablet, mobile).
- AI-Powered Project Summarizer Tool: An AI tool that suggests concise and impactful summary descriptions for projects, based on input keywords or longer project details, to enhance presentation.

## Style Guidelines:

- The visual scheme embraces a sophisticated dark mode. The primary color, chosen to evoke professionalism and creativity, is a soft blue-purple (#A1A1DB).
- The background features a very dark, subtly tinted blue-purple (#26202B), creating an immersive 'archive' feel while maintaining depth.
- An analogous accent color, a bright and clear blue (#A3CBF8), is used to highlight interactive elements and call-to-actions, ensuring good contrast.
- Headlines will utilize 'Space Grotesk' (sans-serif) for a modern, tech-inspired and slightly stylized appearance.
- Body text will use 'Inter' (sans-serif), ensuring clean readability and a neutral, professional tone for descriptions.
- Use minimalist, line-based vector icons for navigation, project categories, and subtle interface elements to maintain a sleek aesthetic.
- Project listings will be presented in a clean, grid-based format with generous spacing, while detailed project pages will adopt a focused, full-width presentation to emphasize content.
- Subtle and elegant hover effects will be applied to interactive components like project cards and navigation links, coupled with smooth page transitions for a fluid user experience.
